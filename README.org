* EASYFOCUS

** Requirements

- [[https://github.com/LemonBoy/bar][lemonbar]]
- wmctrl

*** Optional (EXPERIMENTAL)

* COLOG

Highlight matches

#+BEGIN_SRC sh
colog [<color1> <pattern11> <pattern12> ...] [<color2> <pattern21> <pattern22> ...] ... 
#+END_SRC

** Colors

#+BEGIN_SRC sh
-r|--red
-g|--green
-y|--yellow
-b|--blue
-m|--magenta
-c|--cyan
#+END_SRC

** Patterns

regex patterns.

** Other options

#+BEGIN_SRC sh
-l|--log or without arguments
#+END_SRC

Colorize 'ERROR' with red, 'DEBUG' with green, 'WARN' with yellow, 'INFO' with blue
and numbers with cyan.

#+BEGIN_SRC sh
-s|--sensitive
#+END_SRC

Case sensitive.

** Example

#+BEGIN_SRC sh
echo warn error debug info 10 nil | colog
#+END_SRC

* MINUS
